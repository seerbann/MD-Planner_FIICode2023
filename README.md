## :closed_lock_with_key: Login & Sign In

Doctors can create an account in the app by accessing the "Register as a doctor" button, available on the first page of the application.

![firstPage](https://user-images.githubusercontent.com/72700839/228334007-233fba4a-005f-4379-8564-9436be7039ac.png)
